 COMPLETE EMAIL WORKFLOW (Happy Path)
══════════════════════════════════════════════════════════════════════════════════════════════

  Customer sends email (Dutch)
       │
       ▼
  Gmail Inbox ◄──── Gmail API fetches ◄──── MailFlow Backend
       │
       ▼
  Extract email body (HTML → plain text, strip footers, max 3000 chars)
       │
       ├──▶ Keyword categorize (instant) ──▶ Show in UI immediately
       │
       └──▶ AI categorize (async, batched) ──▶ Update category in UI later
       │
       ▼
  Support Agent selects email in list
       │
       ▼
  Email Detail shows: subject, from, to, date, body, category badge
       │
       ├──▶ [Translate] ──▶ Groq AI detects language ──▶ Shows English translation
       │
       ├──▶ [Mark Done] ──▶ Saves to MongoDB (survives refresh)
       │
       ├──▶ [Flag Callback] ──▶ Add note ──▶ Saves to MongoDB
       │
       └──▶ [Generate AI Reply]
              │
              ▼
         Groq AI creates English draft
              │
              ▼
         Agent reviews & edits draft
              │
              ▼
         Click "Translate & Send"
              │
              ├──▶ Detect customer language (AI)
              └──▶ Translate reply to customer language (AI)
              │
              ▼
         Preview translated reply (editable)
              │
              ▼
         Click "Send"
              │
              ▼
         POST /api/send ──▶ Gmail API sends email
              │               (with In-Reply-To threading)
              ▼
         ✓ Customer receives reply in their language

══════════════════════════════════════════════════════════════════════════════════════════════
 ROLE-BASED ACCESS
══════════════════════════════════════════════════════════════════════════════════════════════

  ┌──────────────┬──────────────────────────────────────────────────────────────┐
  │     ROLE     │  PERMISSIONS                                                │
  ├──────────────┼──────────────────────────────────────────────────────────────┤
  │   Owner      │  Everything: read emails, reply, translate, send,           │
  │              │  manage team (change roles, remove, reactivate members)      │
  ├──────────────┼──────────────────────────────────────────────────────────────┤
  │   Manager    │  Same as Owner: full email access + team management          │
  ├──────────────┼──────────────────────────────────────────────────────────────┤
  │   Agent      │  Read emails, reply, translate, send, mark done/callback     │
  │              │  CANNOT manage team members or change roles                  │
  └──────────────┴──────────────────────────────────────────────────────────────┘

  New member joins: Sign in with Google → auto-created as "Agent"
                    → Owner/Manager promotes to "Manager" if needed
